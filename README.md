¿Qué es Cr4Zy B10? El Manifiesto del Caos
Cr4Zy B10 es un juego anti-intuitivo de formato corto (Nivel Intermedio) desarrollado en un único archivo web. Su objetivo principal es subvertir las expectativas del usuario utilizando mecánicas de engaño, un diseño estético heredado del glitch art y un ritmo frenético.

En lugar de guiar al usuario de forma amigable, la interfaz se convierte en el enemigo.

Los 3 Pilares del "Sin Sentido" (Cómo te rompe la cabeza)
Para lograr que el proyecto sea un caos total, se apoya en tres elementos programados directamente en el código:
La Interfaz Esquiva (Menú Psicodélico): El usuario no puede hacer clic tranquilamente para empezar. Los botones del menú se mueven solos, rebotando por la pantalla como salvapantallas locos de los años 90. Tienes que "cazar" el botón para poder jugar.
La Psicología Inversa (El Engaño): El juego rompe la regla de oro del diseño web: la honestidad. Si haces clic en el botón que promete un juego relajante de ritmo, la web te lanza sin avisar a un clicker estresante donde todo parpadea. El orden lógico está completamente destruido.
Sobrecarga Sensorial (Estética Epiléptica/Glitch): Usando animaciones CSS ultra rápidas, el fondo cambia de colores flúor, los textos vibran como si la pantalla se estuviera rompiendo y los elementos se teletransportan. Es un asalto visual que obliga al jugador a concentrarse el doble.
